Natural Example Application using Templates

Natural Version 6 on open system platforms, adds new functionality for templates and XML stylesheets to the Web interface (see: Online Documentation).

The examples show the usage of the new interfaces to build a web application. Again the example programs works with the employee data stored in ADABAS.

New Design

The example program design was implemented using only one single template (d6-template.html). The template contains several parts ($...$ strings). These strings will be replaced during runtime with the application data. The template itself makes use of css (cascading style sheets) that will be included (<!CSS>) inline to the page (d6-css.css). For better performance this stylesheet can be saved at the http server and call with an reference, by changing the replacement to another file containing the specific link.

Part one: List and Browse

The two menu items Browse and List use internally the same routine (d6emxml) generating an XML stream for a given selection. The different look will be done with different stylesheet transformations one for the listing (d6-list.xsl) and one for the browse (d6-browse.xsl).

Part two: Show, Change, Delete and New

These commands deal only with a single set of data. example use two different html templates for implementation: one for only showing data (d6-output.html), and one for modifying data (d6-input.html). All relevant parts that need to be changed are marked with $...$ strings. For the delete dialog a replacement is defined (<!SUBMIT>) that will be replaced with the delete button...

Alternative Template

A second template (d6-template2.html) has been included to the demo, using a different css (d6-css2.css) to show the flexibility of templates. The generation of the selection (d6-select-left2.html) needs to be different because it is part of the general layout.
Switch between the different templates by changing the variable STYLE2 at d6index and stow.

The Natural Part

The only program that can be called directory from the internet is d6index, with is used to dispatch all request to the different program parts and load all templates and data.

All sources start with d6 and can be found in the library SYSWEB3.
