The SYSUTIL Message Retrieval is used to read Natural or user defined messages. It is called in the following way:

    SYSUTIL -MSG [nnnn [library]]

whereby

    nnnn is the message number

and

    library is the library containing the user defined message.

If the message number and the library are specified, the user defined message nnnn from library library is displayed. If only the message number is specified, the Natural message nnnn is displayed. If no parameter is given, the Natural message NAT0001 is shown initially.

In the tool, you can specify the message type (Natural or user defined), the language of the message, the library of a user defined message and the message number. The tool provides functions to read the previous or next message. For any message it shows the corresponding short and long text. Additionally, the tool provides functions to print the message or to convert it into a PDF document.
